# MVP-IQ Platform Rollout Plan

## Overview
We're rolling out the platform in three strategic versions to test, gather feedback, and iterate before expanding features.

---

## V1: Students/Individuals Only üéØ

**Focus:** Core feedback and mentorship experience for individual players

### Player User Flow
1. **Registration & Onboarding**
   - Sign up as a player
   - Complete profile (name, email, optional photo)
   - Set up account

2. **Video Submission for Feedback**
   - Upload game footage (under 60 seconds, under 1GB)
   - Select which mentor they want feedback from
   - Pay $50 for professional feedback
   - Receive confirmation email

3. **Receiving Feedback**
   - Get email notification when feedback is ready
   - View detailed feedback in dashboard
   - See mentor's analysis and recommendations

4. **One-on-One Sessions**
   - Browse available mentor time slots
   - Book a 1-on-1 session ($150)
   - Receive session confirmation
   - Attend scheduled session
   - If mentor cancels, receive credit to reschedule for free

5. **Account Management**
   - Update profile information
   - Change email, name, phone number
   - Update profile photo
   - View submission history

### Mentor User Flow
1. **Dashboard Overview**
   - See new submissions assigned to them
   - View upcoming one-on-one sessions
   - Quick access to pending feedback requests

2. **Providing Feedback**
   - Receive email notification for new submission
   - Review player's video and notes
   - Submit detailed feedback and analysis
   - Mark feedback as complete

3. **Managing Availability**
   - Set recurring weekly availability slots
   - Add one-time availability slots
   - Manage calendar for one-on-one sessions

4. **One-on-One Sessions**
   - View all scheduled sessions
   - See player information and session details
   - Cancel sessions (triggers credit to player + reschedule link)

### Admin User Flow
1. **User Management**
   - View all users (players, mentors)
   - Edit user information
   - Activate/deactivate accounts
   - Reset passwords

2. **System Oversight**
   - Monitor submissions and sessions
   - View user statistics
   - Manage platform settings

## V2: Add Coaches/Schools üè´

**Focus:** Team management and bulk video submissions for coaches

### Everything from V1, plus:

### Coach User Flow
1. **Team Setup**
   - Create team/school profile
   - Add players to roster
   - Manage player information (name, number)
   - Remove players from team

2. **Video Submissions**
   - Upload videos for team players (15 per month included)
   - Select mentor for each submission
   - Track all team submissions in dashboard
   - View feedback for all players

3. **Team Management**
   - View all players on roster
   - See which players have active submissions
   - Monitor team feedback status

### Player User Flow (Enhanced)
- If player is on a team:
  - Team appears on their individual dashboard
  - Can see team-related submissions
  - Individual account privileges remain ($50 feedback, $150 one-on-ones)
  - If removed from team, team info disappears from dashboard

---

## V3: "Cameo of Sports Mentorship" üåü

**Focus:** Enhanced mentor experience with personalized offerings and expanded mentorship tools

### Everything from V1 & V2, plus:

### Enhanced Mentor Features
1. **Personalized Mentor Profiles**
   - Custom mentor bio and background
   - Highlight achievements and expertise
   - Showcase previous feedback examples (anonymized)
   - Set pricing for different service tiers

2. **Advanced Feedback Tools**
   - Video annotation capabilities
   - Drawing/play diagram tools
   - Multi-format feedback (text, video, audio)
   - Template library for common feedback types

3. **Mentorship Packages**
   - Create custom mentorship packages
   - Offer subscription-based mentorship
   - Bundle services (e.g., "5 feedback reviews + 2 one-on-ones")
   - Set availability for different service levels

4. **Mentor Analytics**
   - View submission completion rates
   - Track average response time
   - See player satisfaction metrics
   - Monitor earnings and session bookings

5. **Direct Communication**
   - In-app messaging with players
   - Follow-up question handling
   - Pre-session preparation tools
   - Post-session notes and recommendations

### Enhanced Player Experience
1. **Mentor Discovery**
   - Browse mentor profiles
   - Filter by expertise, sport, position
   - Read mentor reviews and ratings
   - Compare mentor offerings

2. **Flexible Booking**
   - Book mentors directly from profiles
   - See mentor-specific availability
   - Request custom time slots
   - Gift sessions to other players

3. **Enhanced Feedback Experience**
   - Interactive feedback viewer
   - Video playback with mentor annotations
   - Download feedback reports
   - Share feedback with coaches/parents
